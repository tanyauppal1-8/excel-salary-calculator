# 📊 Data Science Salary Calculator Dashboard

## Overview

The Data Science Salary Calculator Dashboard is an interactive Excel-based analytics solution designed to estimate salaries for Data Science professionals based on selected criteria. The dashboard transforms raw salary data into actionable insights through a clean and user-friendly interface.

Users can explore salary trends by selecting different parameters and instantly receive salary estimates, making the tool useful for salary benchmarking and market analysis.

---

## Business Problem

Salary expectations vary significantly across job roles, countries, and employment types. Accessing relevant compensation insights can be challenging without a structured analytical tool.

This dashboard simplifies salary exploration by providing an interactive platform that enables users to analyze and compare salary information efficiently.
